# DB2 User API (Node.js + Express + IBM DB2)

##Overview

This is a simple REST API built with **Node.js** and **Express** that connects to an **IBM DB2** database using the `ibm_db` package.

The application:

- Connects to a DB2 database on server startup
- Provides basic endpoints to create and fetch users
- Demonstrates embedded SQL queries

---

## 🛠 Tech Stack

- Node.js
- Express.js
- IBM DB2
- ibm_db (IBM native CLI driver)
